# PateryMenu
**Status** : `BETA`
## **Infomation**
A simple patery ui menu components

## **Why use PateryButton?**
Using `PateryMenu` is a wise choice because it's simplified and you can quickly call it right now.

## Usage

```jsx
<PateryMenu component={[
  {element: <PateryButton>Home</PateryButton>},
  {element: <button>Hello</button>}
]}/>
```
